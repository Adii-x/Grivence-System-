# Nexus

**Student Grievance Management System**

Nexus is a platform designed to bridge the gap between students and administration. It allows students to submit, track, and resolve grievances transparently, while providing staff and admins with the tools they need to manage them efficiently.

## Core Features
- **Role-based Access** (Student, Staff, Admin)
- **Complaint Tracking** & Auto-Escalation
- **Admin Dashboards** & Department Analytics
- **Modern UI** with dark theme

## Tech Stack
- React 18 + Vite
- TypeScript
- Tailwind CSS + Radix UI
- Zustand (State Management)
- Recharts (Analytics Data)

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
